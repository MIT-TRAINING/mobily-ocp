Concept · Configuration & storage

Configure without rebuilding. Persist beyond the pod.

A good image is immutable — the same one runs everywhere. ConfigMaps and Secrets inject what differs at runtime (but a Secret is only base64, not encrypted). A PVC gives data a lifecycle independent of any pod, so it survives deletion.

PVC: data outlives the pod

A WaitForFirstConsumer PVC stays Pending until a pod mounts it. Write a CDR, destroy the pod, then mount a new pod on the same claim.
